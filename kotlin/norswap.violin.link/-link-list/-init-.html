LinkList(vararg items: T)

Builds a mutable linked list from items.

The items are inserted right-to-left (so the last item is in the innermost link, and iteration order is order in which the items appear).





LinkList(link: Link<T>? = null, size: Int = link.iterator().stream().count())

A mutable singly linked list implemented as a pointer to an immutable linked list (Link) and a size.

size must be the real size of link.

The equals method is shallow: two link lists are equal if their size are the same and their link are reference-equals.
