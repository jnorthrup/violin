LinkList

class LinkList<T : Any> : Stack<T>, Cloneable

A mutable singly linked list implemented as a pointer to an immutable linked list (Link) and a size.

size must be the real size of link.

The equals method is shallow: two link lists are equal if their size are the same and their link are reference-equals.
