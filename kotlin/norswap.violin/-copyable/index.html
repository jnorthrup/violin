Copyable

interface Copyable : Cloneable

An interface analogous to the Kotlin version of Cloneable, except that copy is public and that copycast is available for effortless casting.



Functions

copy open fun copy(): Any

Returns a copy of the object.
copycast open fun <T : Any> copycast(): T

Same as copy, but casts the copy to T (which should be inferred, use an explicit cast otherwise).

Extension Functions

cast fun <T> Any?.cast(): T

Casts the receiver to T.
