minWith

inline fun <T : Any> Stream<T>.minWith(cmp: (T, T) -> Int): T?

Return the minimum item of the stream according to cmp (same contract as java.util.Comparator.compare). If two items compare identical, the earliest will be preferred.



fun <T : Any> Stream<T>.minWith(cmp: Comparator<T>): T?

Return the minimum item of the stream, determined by delegation to cmp. If two items compare identical, the earliest will be preferred.
